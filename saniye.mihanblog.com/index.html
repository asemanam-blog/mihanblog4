مشکلات راه اندازی JDK
آموزش اندروید بصورت تجربی/ JDK رو نصب کردم و بعد Android Studio
اما هنگام اجرای اندروید استودیو، اروری مشاهده شد که JDK رو پیدا نمیکنه
پس از سرچ در اینترنت، گزینه های زیادی برای رفع این مشکل مشاهده شد. اما هیچ کدوم نتونست مشکل رو رفع کنه. نهایتا به یک راه حل ترکیبی رسیدم:

برای آنکه سیستم شما JDK را شناسایی کند لازم است که به صورت زیر عمل نمایید:

Start -> Control Panel -> System -> Advanced System Settings

بر روی تب Advanced کلیک نمایید و دکمه ی Envioronment Variables را انتخاب کنید.

در پنجره ی ظاهر شده در قسمت بالای این پنجره User Variables For User بر روی دکمه New کلیک کنید. برای Variable Name نام JAVA_HOME را تایپ نمایید و برای Variable Value مسیر نصب JDK  را وارد نمایید و بر روی دکمه OK کلیک کنید ( آدرس بدون مسیر دایرکتوری Bin )
به طور مثال:

\C:\Program Files\Java\JDK1.7.0_5

(حتما \ رو انتهای آدرس بذارید)

حال باید دوباره در پنجره Envioronment Variables تغییراتی انجام دھیم. قسمت پایین این پنجره System Variables به دنبال Variable ای به نام Path بگردید. سپس آن را انتخاب کرده و دکمه Edit را کلیک کنید و در Variable Valu آدرس دایرکتوری Bin درون JDK را اضافه کنید.
به طور مثال:

C:\Program Files\Java\JDK1.7.0_51\bin\

(حتما \ رو انتهای آدرس بذارید)

در نظر داشته باشید که برای اضافه کردن یک آدرس به Path باید بعد از ھر آدرس علامت نقطه ویرگول ; اضافه کنیم

حال مراحل نصب JDK با موفقیت به اتمام رسیده است.

برای اطمینان خاطر از نصب Java وارد محیط خط فرمان یا Command Prompt شوید

و دستورات javac -version و java -version را وارد نمایید. اگر مراحل نصب را به درستی طی کرده باشید باید پاسخی مناسب دریافت کنید.

یک نقاشی باور نکردنی از چهره مورگان فریمن؛ شاهکاری که با نوک انگشت بر روی آیپد کشیده شده
دنیا چندان نقاشی با انگشت روی آیپد را جدی نگرفته ولی هنرمند آیپد نگار، آقای کایل لامبرت که قبلا با نقاشی های فوق العاده اش همه را شگفت زده کرده بود، حالا برگشته تا موهای تن مان را سیخ کند.

اگر ساعت ها برای شما دلیل بیاوریم که آنچه در بالا می بینید، یک نقاشی است نه عکس، تا خودتان فیلم کوتاه مراحل کشیده شدن آن را نبینید، باور نخواهید کرد.

نقاشی فوق العاده کایل از چهره بازیگر مشهور، مورگان فریمن، حاصل ۲۰۰ ساعت کار است و با ۲۸۵,۰۰۰ بار لمس سر انگشتان او در محیط اپلیکیشن Procreate کشیده شده.

برای اینکه حس درستی از میزان جزئیات خلق شده در این نقاشی به دست بیاورید، کایل یک ویدیوی گذر زمان نیز تهیه کرده که فرآیند خلق آن را به تصویر می کشد. این شما و این هم روند کشیدن این نقاشی سر انگشتی.
