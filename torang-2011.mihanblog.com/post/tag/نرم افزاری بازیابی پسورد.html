Password Recovery Kit نرم افزاری برای بازیابی پسورد است در واقع بوسیله این نرم افزار میتوانید رمز های عبور معروف ترین برنامه های جهان از جمله تمامی نرم افزار های مایکروسافت آفیس و FileMaker, Outlook, Outlook Express, Exchange, WinZip, PKZip, ZIP, WinRAR, Network Connections, SQL, BestCrypt, OneNote, PowerPoint, VBA Visual Basic modules و ... را بازیابی کنید. همچنین بدست آوردن رمزهای ویندوز و اینترنت اکسپلور و نرم افزار های سیستمی از ویژگی های دیگر این نرم افزار از است.
به کمک نرم افزار میتوانید رمز عبور 180 فرمت مختلف را بازیابی کنید و همچنین تعداد کارکتر های امنیتی و استفاده از کارکتر های خاص برای این نرم افزار هیچ اهمیتی ندارد که میتوانیم آن را از قابلیت های مهم این نرم افزار برشماریم. برنامه بازیابی رمز عبور Password Recovery Kit از قابلیت TACC سخت افزاری استفاده میکند که سرعت بازیابی رمز را تا 25برابر افزایش میدهد. این نرم افزار سازگاری کاملی با مجموعه آفیس دارد طوری که حمله 5 گانه این امکان را میدهد تا رمز اسناد و پروژه های آفیس را برملا کنید.
